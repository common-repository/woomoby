== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woomoby` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the WooMoby screen to configure the App.
